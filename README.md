# WaterTracker
A simple Python project that helps users track their daily water intake and monitor hydration goals.

---

Features

- Set a daily water intake goal
- Record morning, afternoon, and evening water consumption
- Calculate total water intake
- Calculate goal completion percentage
- Display remaining water needed
- Show hydration status

---

Technologies Used

- Python

---

Concepts Used

This project uses:

- Variables
- User Input
- Arithmetic Operations
- Conditional Statements
- Percentage Calculation
- Output Formatting

---

Installation

1. Install Python from https://www.python.org/
2. Download or clone this repository.
3. Open a terminal in the project folder.

---

How to Run

python main.py

---

How It Works

1. User enters a daily water goal in milliliters (ml).
2. User enters water intake for:
   - Morning
   - Afternoon
   - Evening
3. Program calculates:
   - Total water intake
   - Goal completion percentage
   - Remaining water required
4. Program displays hydration status.

---

Formula Used

Total Water Intake

Total = Morning + Afternoon + Evening

Goal Completion Percentage

Percentage = (Total Water Intake / Daily Goal) × 100

---

Hydration Status

Percentage| Status
100% or more| Excellent
75% – 99%| Good
50% – 74%| Average
Below 50%| Drink More Water
